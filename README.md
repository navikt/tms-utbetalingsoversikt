# tms-utbetalingsoversikt 
Frontend for utbetalingsoversikten.


## Komme i gang

For å kjøre opp appen lokalt:

1. `npm install`
1. `npm run dev`
1. Appen nås på http://localhost:3000/utbetalingsoversikt
